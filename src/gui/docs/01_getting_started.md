# Быстрый старт

## Установка

SK24mapper распространяется как портативное приложение. Распакуйте архив в удобную директорию и запустите `SK24mapper.exe`.

### API-ключ Mapbox

Для загрузки спутниковых снимков и данных рельефа необходим токен Mapbox:

1. Зарегистрируйтесь на [mapbox.com](https://www.mapbox.com/)
2. Создайте токен в разделе *Account → Tokens*
3. В приложении: **Файл → API ключ...** — вставьте токен

Токен сохраняется локально в файле `.secrets.env` и не передаётся третьим сторонам.

## Создание первой карты

1. Введите координаты центра карты в поля **X** и **Y** (в системе СК-42)
2. Выберите **масштаб** (1:25000, 1:50000, 1:100000 и т.д.)
3. Выберите **тип карты** из выпадающего списка
4. Нажмите кнопку **Создать карту**
5. После генерации нажмите **Сохранить карту** для экспорта в JPEG

## Интерфейс

### Левая панель

- **Координаты** — поля ввода X (северное) и Y (восточное) в метрах СК-42
- **Масштаб** — выбор масштаба карты
- **Тип карты** — выбор стиля визуализации (спутник, гибрид, рельеф и т.д.)
- **Параметры** — дополнительные настройки для выбранного типа карты

### Правая панель

- **Предпросмотр** — интерактивная область просмотра карты
- Курсор мыши показывает координаты в реальном времени
- Правый клик — перемещение контрольной точки

### Строка состояния

Отображает координаты курсора (СК-42 и WGS-84), высоту рельефа и состояние загрузки.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `F1` | Открыть документацию |
| `Ctrl+N` | Новый профиль |
| `Ctrl+O` | Открыть профиль |
| `Ctrl+S` | Сохранить профиль |
| `Ctrl+Shift+S` | Сохранить профиль как... |
| `Ctrl+Q` | Выход |
