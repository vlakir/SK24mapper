# Дополнительные настройки

## Параметры Гельмерта

Преобразование между WGS-84 и СК-42 выполняется по 7-параметрической модели Гельмерта. Стандартные параметры (Госстандарт) обеспечивают точность ±2–5 м на территории РФ.

Для повышения точности в конкретном регионе можно задать локальные параметры Гельмерта:

**Операции → Дополнительные параметры → Параметры Гельмерта**

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| $\Delta X$ | Сдвиг по оси X (м) | 23.92 |
| $\Delta Y$ | Сдвиг по оси Y (м) | −141.27 |
| $\Delta Z$ | Сдвиг по оси Z (м) | −80.9 |
| $\omega_x$ | Вращение вокруг X (") | 0 |
| $\omega_y$ | Вращение вокруг Y (") | −0.35 |
| $\omega_z$ | Вращение вокруг Z (") | −0.82 |
| $m$ | Масштаб (×10⁻⁶) | −0.12 |

## Качество JPEG

Качество выходного JPEG-файла можно настроить в диапазоне 50–100%. По умолчанию — 92%.

Более высокое качество увеличивает размер файла, но сохраняет детализацию карты. Для печати рекомендуется 95%, для экранного просмотра — 85–92%.

## Километровая сетка

Карта генерируется с наложением километровой сетки в проекции Гаусса-Крюгера. Параметры сетки:

- **Шаг** определяется масштабом карты (1 км для 1:25000, 2 км для 1:50000 и т.д.)
- **Подписи** — координаты в километрах по осям X и Y
- **Цвет и толщина** — настраиваются автоматически для оптимальной читаемости

## Профили настроек

Профили позволяют сохранять и загружать наборы параметров (координаты, масштаб, тип карты, параметры Гельмерта).

### Операции с профилями

| Действие | Горячая клавиша |
|----------|----------------|
| Новый профиль | `Ctrl+N` |
| Открыть профиль | `Ctrl+O` |
| Сохранить профиль | `Ctrl+S` |
| Сохранить как | `Ctrl+Shift+S` |

Профили хранятся в формате TOML в директории:
- Windows: `%APPDATA%\SK42mapper\profiles\`
- Linux: `~/.config/SK42mapper/profiles/`

### Формат профиля

```toml
[coordinates]
x = 5731234.0
y = 7456789.0

[map]
scale = "1:50000"
type = "HYBRID"

[helmert]
dx = 23.92
dy = -141.27
dz = -80.9
```

## Кэш тайлов

Загруженные тайлы кэшируются локально для ускорения повторных запросов:
- Windows: `%LOCALAPPDATA%\SK42mapper\.cache\`
- Linux: `~/.cache/SK42mapper/`

Кэш можно очистить вручную, удалив содержимое директории.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `F1` | Документация |
| `Ctrl+N` | Новый профиль |
| `Ctrl+O` | Открыть профиль |
| `Ctrl+S` | Сохранить профиль |
| `Ctrl+Shift+S` | Сохранить профиль как... |
| `Ctrl+Q` | Выход |
| `Shift+Колёсико` | Вращение азимута РЛС (в режиме РЛС) |
| Правый клик | Установить контрольную точку |
