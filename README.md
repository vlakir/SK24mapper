# SK42

Приложение для создания топографических карт с километровой сеткой в системе координат СК-42 / Гаусса-Крюгера.

## Типы карт

### Спутниковые и картографические основы

- **Спутник** — спутниковые снимки высокого разрешения
- **Гибрид** — спутниковые снимки с наложением дорог и подписей
- **Улицы** — схематическая карта дорожной сети
- **Топографический** — топографическая карта с рельефом и тропами

### Карты высот

- **Карта высот (цветовая шкала)** — раскраска рельефа по высоте с цветовой легендой
- **Карта высот (контуры)** — изолинии рельефа (изогипсы) с подписями высот
- **Карта высот (hillshade)** — теневая отмывка рельефа (планируется)

### Специальные карты

- **Радиогоризонт НСУ БпЛА** — зона радиовидимости наземной станции управления с учётом рельефа местности. Настраиваемая высота антенны, диапазон высот целей, режим отсчёта высоты (от контрольной точки, от земли, от уровня моря). Интерактивный пересчёт при изменении параметров.
- **Зона обнаружения РЛС** — зона обнаружения радиолокационной станции с учётом рельефа. Настраиваемая высота антенны, мощность и параметры диаграммы направленности.

## Общие возможности

- **Километровая сетка СК-42/ГК** с подписями координат на всех типах карт
- **Изолинии рельефа** — опциональное наложение на любой тип карты
- **Контрольные точки** для привязки и проверки точности позиционирования
- **Настраиваемые параметры Гельмерта** для точного преобразования WGS-84 ↔ СК-42
- **Профили конфигурации** — сохранение и загрузка наборов настроек
- **Экспорт** готовых карт в графические файлы

## Установка и запуск

Требуется ключ Mapbox API. Задайте его через меню приложения или в файле `.env`:

```
API_KEY=ваш_mapbox_access_token
```

```bash
poetry install
poetry run python src/main.py
```

## Данные приложения

- **Профили и карты**: `%APPDATA%/SK42/`
- **Кэш тайлов**: `%LOCALAPPDATA%/SK42/.cache/tiles/`
- **Логи**: `%LOCALAPPDATA%/SK42/log/`
